Binary Clock:

This is a binary clock I made from scratch. I designed the clock, wrote the software, and etched the PCB board.

Materials:

1x Atmel ATMega328, 2x 10uF Capacitor, 2x 22pF Capacitor, 10x 220ohm Resistor, 1x 1kohm Resistor, 2x 10kohm Resistor, 1x 16Mhz XTAL, 3x Push Button Switch, 1x Power Jack, 1x 5v Regulator

Software & Languages:

Arduino Processing, CadSoft EAGLE PCB Design Software
